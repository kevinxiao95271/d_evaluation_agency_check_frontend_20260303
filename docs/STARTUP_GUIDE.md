# 机构考核评估系统 - 启动指南

## 🚀 快速启动

### 方法一：使用Node.js开发服务器（推荐）

1. **安装依赖**
```bash
npm install
```

2. **启动开发服务器**
```bash
npm run dev
```

3. **访问应用**
- 前端地址：http://localhost:5039
- 后端API：http://localhost:5031

### 方法二：使用静态HTML测试（临时方案）

如果Node.js环境有问题，可以直接打开 `test.html` 文件进行API测试：

1. 确保后端服务运行在 http://localhost:5031
2. 用浏览器打开 `test.html` 文件
3. 点击各个测试按钮验证API功能

## 🔧 开发环境配置

### 代理配置
前端开发服务器已配置API代理：
```javascript
// vite.config.js
server: {
  port: 5039,
  proxy: {
    '/api': {
      target: 'http://localhost:5031',
      changeOrigin: true
    }
  }
}
```

### 跨域处理
所有API请求通过Vite代理转发，无需额外配置CORS。

## 🐛 常见问题

### 1. Node.js环境问题
**问题**: 找不到node或npm命令
**解决**: 
- 检查Node.js是否正确安装
- 确认环境变量PATH包含Node.js路径
- 重启终端或IDE

### 2. 端口冲突
**问题**: 5039端口被占用
**解决**: 
- 修改 `vite.config.js` 中的端口号
- 或者停止占用端口的其他服务

### 3. API连接失败
**问题**: 无法连接后端API
**解决**: 
- 确认后端服务运行在5031端口
- 检查防火墙设置
- 使用test.html直接测试API

### 4. 依赖安装失败
**问题**: npm install报错
**解决**: 
- 清除缓存：`npm cache clean --force`
- 删除node_modules重新安装
- 使用yarn替代npm
