# 机构考核评估系统 - 项目总结

## 🎯 项目概述

基于Vue 3 + Element Plus开发的机构考核评估系统前端，支持专家评委和大众评委对各机构进行评分，并提供完整的管理后台功能。

## 🚀 功能特性

### 评委端功能
1. **登录认证** - 支持评委身份验证
2. **待评任务** - 显示需要评分的机构列表
3. **双模式评分**:
   - 直接打总分：输入总分，系统按权重自动分配
   - 逐条打分：对每个评分条目单独评分
4. **评分进度** - 实时显示评分完成状态

### 管理端功能
1. **机构管理** - 机构信息的增删改查
2. **评委管理** - 评委账号的维护管理
3. **任务管理** - 考核任务的创建和配置
4. **评分表配置** - 评分分类和条目的设置
5. **附加项管理** - 主任演讲、秘书参与加分
6. **系统配置** - 评分规则和权重设置
7. **结果统计** - 评分结果查看和排名

### 监控工具
1. **评分状态总览** - 实时查看所有评委评分进度
2. **API测试工具** - 在线测试后端接口
3. **账号快速切换** - 便于测试不同评委视角

## 🎮 使用流程

### 评委评分流程
1. 使用评委账号登录 (expertX/123456 或 publicX/123456)

3. 选择机构进入评分页面

3. 点击评委账号快速切换测试
4. 实时监控评分完成情况

## 📊 评分算法

### 评分权重分配
- **专家评委权重**: 70%
- **大众评委权重**: 20%
- **附加项权重**: 10%

### 计算公式
```
最终得分 = (专家平均分 × 0.7 + 大众平均分 × 0.2) × 系统总分/模板总分 + 附加分

其中：
- 专家平均分：去掉最高最低分后的平均值
- 大众平均分：去掉最高最低分后的平均值  
- 系统总分：45分
- 模板总分：100分
- 附加分：主任演讲(8分) + 秘书参与(2分)
```
